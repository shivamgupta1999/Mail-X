# Mail-X

# Tech Stack
- ReactJs
- NodeJS
- Express
- MongoDB

Deployed on Heroku(Backend) and Netlify(Frontend).

# Running the Backend

## Build and run
  1.Clone Repo:

    git clone https://github.com/shivamgupta1999/Mail-X.git
    
  2.To install all the dependencies, enter command:
    
    npm install
    
  3.To run:
    
    npm run dev


# Running Frontend

## Build and run
    
  1.To install all the dependencies, enter command:
    
    npm install
    
  2.To run:
    
    npm start
